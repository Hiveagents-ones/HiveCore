# 教练排班系统用户手册

## 1. 功能概述
教练排班系统用于管理健身房教练的工作安排，包括：
- 查看教练排班表
- 添加新的排班记录
- 修改已有排班
- 删除排班记录
- 管理教练请假申请

## 2. 使用说明

### 2.1 访问排班界面
1. 登录系统后，从左侧导航菜单选择"教练排班"
2. 系统将显示默认日期（当天）的排班表

### 2.2 查看排班
1. 使用日期选择器选择要查看的日期
2. 系统将自动刷新显示所选日期的排班信息
3. 排班表包含以下信息：
   - 教练姓名
   - 课程名称
   - 开始时间
   - 结束时间

### 2.3 添加排班
1. 点击"新增排班"按钮
2. 在弹出的对话框中填写以下信息：
   - 选择教练（从下拉列表）
   - 选择课程（从下拉列表）
   - 设置开始时间（日期和时间）
   - 设置结束时间（日期和时间）
3. 点击"确认"按钮保存

### 2.4 修改排班
1. 在排班表中找到要修改的记录
2. 点击该记录对应的"编辑"按钮
3. 在弹出的对话框中修改相关信息
4. 点击"确认"按钮保存更改

### 2.5 删除排班
1. 在排班表中找到要删除的记录
2. 点击该记录对应的"删除"按钮
3. 系统将弹出确认对话框，确认后删除该排班

### 2.6 管理请假
1. 点击导航菜单中的"教练请假"
2. 查看当前请假申请列表
3. 可以批准或拒绝请假申请

## 3. 注意事项
1. 排班时间不能重叠
2. 删除排班前请确保没有会员已预约该课程
3. 系统会自动检查时间冲突并提示

## 4. 常见问题

### Q: 为什么无法添加排班？
A: 可能原因包括：
- 选择的时间段已有其他排班
- 教练在该时间段已有请假记录
- 输入的时间格式不正确

### Q: 如何查看特定教练的排班？
A: 目前可以通过以下方式：
1. 在排班表中点击"教练姓名"列进行排序
2. 或者使用浏览器页面搜索功能（Ctrl+F）查找特定教练

### Q: 排班修改后为什么会员看不到？
A: 排班修改后需要约5分钟同步到会员端，如长时间未更新请联系技术支持。
