# Hello World 项目

## 项目概述

本项目是一个简单的静态网页，用于展示 "Hello World" 文本。项目采用纯静态技术栈（HTML5, CSS3, Vanilla JavaScript），无需构建工具或服务器依赖，确保最低的加载延迟和最高的部署简便性。

## 功能实现

### R1: 显示 Hello World 文本

- **描述**: 在网页的主体区域，清晰地展示“Hello World”字符串。
- **实现**: 通过 `app.html` 文件中的 HTML 结构和 CSS 样式实现。
- **优先级**: P1

## 部署和使用

### 本地运行

1. 克隆或下载项目文件到本地。
2. 使用浏览器直接打开 `app.html` 文件即可查看页面效果。
3. 页面加载后会自动显示 "Hello World" 文本，并支持以下功能：
   - 主题切换（亮色/暗色）
   - 多语言支持（中文/英文）
   - 响应式布局适配

### 部署到服务器

1. 将项目文件上传到任意静态文件服务器（如 Nginx、Apache 等）。
2. 通过浏览器访问服务器上的 `app.html` 文件即可。
3. 确保服务器支持以下特性：
   - 正确的 MIME 类型设置（.html 文件）
   - 支持 HTTPS（推荐）
   - 启用 Gzip 压缩（可选）
